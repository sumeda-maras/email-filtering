# Spam vs. Ham Email Classifier

## Overview
This project builds a machine learning classifier to distinguish spam emails from legitmate (ham) emails. It applies text preprocessing and feature engineering techniques to train a logistic regression model that predicts the probability of an email being spam.

## Dataset
The model is trained on a labeled email dataset containing:
- Email subject lines
- Email body text
- Spam/Ham labels

## Technologies
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

## Key Metrics
- **Training Accuracy:** **75.8%**
- Spam probability predictions
- Classification accuracy
